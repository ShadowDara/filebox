# Upload Box

a little file upload Server written in Flask and Preact

### Start

to start run
```sh
cd frontend
npm install

npm run build

cd ..

python -m venv venv
venv/scripts/Activate

pip install requirements.txt

python app.py
```

to build run
```sh
python build_export.py
```
